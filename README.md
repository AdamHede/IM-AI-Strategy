# AI AIM Strategy

This project contains the strategic presentation and living wiki for Implement Consulting Group's AI initiative.

## Structure

*   **Presentation**: `index.html` - The main entry point, a slide-based presentation of the strategy.
*   **Wiki**: `wiki/` - The generated HTML documentation of the strategy.
*   **Source**: `strategy_wiki/` - The markdown source files for the strategy wiki.

## Usage

1.  Open `index.html` in your browser to view the presentation.
2.  Navigate through the slides using the arrow keys or on-screen controls.
3.  Click the "Wiki" link to explore the detailed strategy documents.

## Development

To update the strategy wiki:

1.  Edit the markdown files in `strategy_wiki/`.
2.  Run the build script to regenerate the HTML:
    ```bash
    node build.js
    ```
3.  Open `wiki/README.html` to verify changes.
